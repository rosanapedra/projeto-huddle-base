## Meu processo

### Construído com

- Semântica HTML5;
- Propriedades CSS;
- Flexbox;
- Media queries. 

### O que eu aprendi

Aprendi a usar o main no meio entre header e footer, pratiquei o uso dos elementos de flexbox e suas propriedades, além da responsividade. Aprendi também, a usar o hover nos ícones e botão.
